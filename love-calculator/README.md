# ❤️ Love Calculator

A modern and responsive **Love Calculator** built using **HTML, CSS, and JavaScript**. This fun web application generates a random love compatibility percentage between two names while providing an attractive user interface, smooth animations, and a seamless user experience.

> **Disclaimer:** This project is created for **entertainment and educational purposes only**. The generated love percentage is completely random and should not be considered a real compatibility measurement.

---

## ✨ Features

* ❤️ Random Love Percentage Generator
* 🎨 Modern & Romantic User Interface
* 📱 Fully Responsive Design
* ⚡ Smooth CSS Animations
* ⌨️ Keyboard Interaction Support
* 💾 LocalStorage History Tracking
* 🖼️ Custom Background Image
* 🌙 Beginner-Friendly Code Structure
